\textcolor{primary}{\textbf{\Large Executive Summary}}\\[0.3cm]
This report presents a comprehensive Machine Learning pipeline developed to classify Premier League players based on their statistical performance. The objective is to determine whether a player's statistical profile "deserves" a position in a "Top 6" club (Manchester City, Liverpool, Manchester United, Chelsea, Leicester City, and Tottenham Hotspur) using data up to 2020. Two algorithms---Logistic Regression (discriminative) and Linear Discriminant Analysis / GDA (generative)---are compared across four playing positions: Forwards, Midfielders, Defenders, and Goalkeepers.
\end{tcolorbox}
\end{minipage}
\end{center}

\section{Methodology \& Pipeline Overview}

\subsection{Initial Filtering}
\begin{itemize}
    \item \textbf{Appearance Filter:} Only players with at least 10 appearances (\texttt{Appearances >= 10}) are included.
\end{itemize}

\subsection{Data Cleaning \& Preprocessing}
\begin{itemize}
    \item \textbf{String to Numeric Conversion:} Percentage columns stripped of \% symbol.
    \item \textbf{Imputation:} Missing values filled with 0.
    \item \textbf{Integrity Check:} Rows with missing critical data dropped.
\end{itemize}

\subsection{Feature Engineering \& Clipping}
\begin{itemize}
    \item \texttt{Goals per match} capped at 1.2 for forwards.
    \item \texttt{Shooting accuracy \%} capped at 80 for forwards.
    \item \texttt{Passes per match} capped at 90 for midfielders.
\end{itemize}

\subsection{Position-Based Feature Selection}
\begin{table}[H]
\centering
\caption{\textcolor{primary}{\textbf{Features Used by Position}}}
\begin{tabular}{@{}>{\bfseries\color{primary}}p{3cm} p{11cm}@{}}
\toprule
\textbf{Position} & \textbf{Features} \\
\midrule
\rowcolor{lightgray} Forwards & Goals per match, Shooting accuracy \%, Assists, Passes per match, Big chances created \\
Midfielders & Passes per match, Big chances created, Assists, Through balls, Interceptions, Recoveries, Tackles, Cross accuracy \% \\
\rowcolor{lightgray} Defenders & Clean sheets, Tackles, Tackle success \%, Interceptions, Clearances, Aerial battles won \\
Goalkeepers & Saves, Penalties saved, Clean sheets, Goals conceded, High Claims, Punches, Sweeper clearances, Goal Kicks \\
\bottomrule
\end{tabular}
\end{table}

\subsection{Modeling Strategy}
\begin{itemize}
    \item \textbf{Logistic Regression:} \texttt{class\_weight='balanced'}
    \item \textbf{GDA / LDA:} Priors set to [0.5, 0.5]
\end{itemize}

\section{Conclusion \& Recommendations}

\begin{tcolorbox}[colback=lightgray, colframe=primary, arc=5pt, boxrule=1.5pt]
\centering
\textcolor{primary}{\textbf{\Large Summary of Key Findings}}
\end{tcolorbox}

\begin{enumerate}
    \item \textbf{Position Matters:} GDA excels for Forwards (84\%), Midfielders (89\%), Goalkeepers (82\%). Logistic Regression excels for Defenders (86\%).
    \item \textbf{Feature Clipping is Essential:} Prevents outlier distortion.
    \item \textbf{Class Balancing Improves Fairness:} Ensures minority class consideration.
    \item \textbf{Elite Player Validation:} Agüero (87.7\%) and van Dijk (79.1\%) receive strong confidence.
\end{enumerate}
