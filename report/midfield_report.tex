\section{Data Preprocessing \& Feature Engineering - Midfielders}

\subsection{Data Cleaning for Midfielders}

\textbf{First: Player Filtering}

Players were filtered to include only midfielders with at least one appearance. This ensures that the analysis is based on real players who have an actual impact on the pitch.

\textbf{Second: Dropping Irrelevant Columns}

All columns related to defensive performance and goalkeeping were removed, including:
\begin{itemize}
    \item Clean sheets, Goals conceded
    \item Tackles, Tackle success \%, Last man tackles
    \item Blocked shots, Interceptions, Clearances
    \item Headed Clearance, Clearances off line, Recoveries
    \item Duels won, Duels lost, Successful 50/50s
    \item Aerial battles won, Aerial battles lost
    \item Own goals, Errors leading to goal
    \item Saves, Penalties saved, Punches
    \item High Claims, Catches, Sweeper clearances
    \item Throw outs, Goal Kicks
\end{itemize}

The reason for removing these columns is that midfielders are neither pure defensive players nor goalkeepers, so these columns would either be zeros or missing values that add no meaningful insight to the analysis.

\textbf{Third: Feature Engineering}

New features were created to measure midfielder performance more accurately:

\begin{itemize}
    \item \textbf{Passes per Match:} Total passes divided by number of appearances.
    \[
    \text{passes\_per\_match} = \frac{\text{Passes}}{\text{Appearances}}
    \]
    
    \item \textbf{Assists per Match:} Total assists divided by number of appearances.
    \[
    \text{assists\_per\_match} = \frac{\text{Assists}}{\text{Appearances}}
    \]
    
    \item \textbf{Big Chances Created per Match:} Total big chances created divided by number of appearances.
    \[
    \text{big\_chances\_created\_per\_match} = \frac{\text{Big chances created}}{\text{Appearances}}
    \]
    
    \item \textbf{Shots per Match:} Total shots divided by number of appearances.
    \[
    \text{shots\_per\_match} = \frac{\text{Shots}}{\text{Appearances}}
    \]
    
    \item \textbf{Shot Conversion Rate:} Goals divided by shots, with division by zero handled using np.where.
    \[
    \text{shot\_conversion} = \frac{\text{Goals}}{\text{Shots}} \quad \text{(if Shots > 0, else NaN)}
    \]
    
    \item \textbf{Goals per Shot on Target:} Goals divided by shots on target, with division by zero handled using np.where.
    \[
    \text{goals\_per\_shot\_on\_target} = \frac{\text{Goals}}{\text{Shots on target}} \quad \text{(if Shots on target > 0, else NaN)}
    \]
\end{itemize}

\subsection{Validation \& Quality Assurance for Midfielders}

The following steps were performed to ensure data quality for midfielders:

\begin{itemize}
    \item \textbf{Missing Value Handling:} Missing values were handled using conditional statements (np.where), so if a value is zero or missing, the new feature is not calculated.
    \item \textbf{Percentage Validation:} All categorical percentage distributions were verified to sum to 100\% within rounding tolerance.
    \item \textbf{Descriptive Statistics:} The describe() method was applied to validate data distributions and identify outliers.
    \item \textbf{Data Types:} Numerical columns were verified to ensure correct data types for statistical analysis.
\end{itemize}

    \item \textbf{Crossing Accuracy:} Eberechi Eze and Nampalys Mendy share the highest cross accuracy (67\%).
    
    \item \textbf{Through Ball Specialists:} Jack Wilshere's 156 through balls indicate exceptional line-breaking ability.
    
    \item \textbf{Long Ball Proficiency:} Mark Noble's 1,484 accurate long balls demonstrate his deep-lying playmaker role.
    
    \item \textbf{Goal Scoring Midfielders:} Gylfi Sigurdsson leads midfielders with 61 goals.
\end{enumerate}

\vspace{0.3cm}

\begin{tcolorbox}[colback=primary!10, colframe=primary, arc=5pt, boxrule=1.5pt]
\textcolor{primary}{\textbf{Final Recommendations:}}
\begin{itemize}
    \item Prioritize midfielders with high assists per match (above 0.40) for creative roles
    \item Use big chances created as a leading indicator for potential assist production
    \item Balance the midfield with passer, creator, and finisher profiles
    \item Monitor disciplinary records to avoid suspensions
    \item Invest in players with high cross accuracy (above 60\%) for wide positions
\end{itemize}
