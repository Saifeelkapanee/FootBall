\section{Data Preprocessing \& Feature Engineering}


Prior to analysis, the following preprocessing steps were performed:

\begin{itemize}[leftmargin=1.2cm, label=\textcolor{primary}{\faCheckCircle}]
    \item \textbf{Missing Values:} All missing values (NA) were imputed using median imputation for numerical features and mode imputation for categorical features.
    \item \textbf{Feature Engineering:} New features were created including:
        \begin{itemize}
            \item Goals per Match (GPM)
            \item Shot Conversion Rate (Goals / Shots)
            \item Shooting Accuracy (Shots on Target / Total Shots) × 100
            \item Goals per Shot on Target
        \end{itemize}
    \item \textbf{Column Dropping:} Irrelevant columns (player ID, season indicators, redundant match statistics) were removed.
    \item \textbf{Validation:} All categorical percentage distributions were verified to sum to 100\%.
\end{itemize}

\section{Conclusion \& Recommendations}

\begin{tcolorbox}[colback=lightgray, colframe=primary, arc=5pt, boxrule=1.5pt]
\centering
\textcolor{primary}{\textbf{\Large Summary of Key Findings}}
\end{tcolorbox}

\vspace{0.5cm}

\begin{tcolorbox}[colback=lightgray, colframe=secondary, arc=5pt, boxrule=1pt]
The comprehensive analysis of forward player performance reveals several critical insights that can inform team strategy, player recruitment, and training focus.
\end{tcolorbox}

\vspace{0.3cm}

\begin{enumerate}[leftmargin=1.2cm, label=\textcolor{accent}{\textbf{\arabic*.}}]
    \item \textbf{Goal Scoring Elite:} A small percentage of forwards account for the majority of goals scored, confirming the importance of investing in top-tier goal scorers.
    
    \item \textbf{Age-Performance Relationship:} Peak goal-scoring performance occurs between ages 21–25, with age 23 showing the highest goals per match (5.0). This suggests clubs should prioritize players in this age window.
    
    \item \textbf{Efficiency Matters:} Players like Steven Bergwijn demonstrate that high shot conversion (33.3\%) can compensate for fewer total attempts, highlighting the value of clinical finishers.
    
    \item \textbf{xG as a Tool:} The xG difference bubble chart effectively identifies overperformers (positive difference) and underperformers (negative difference), serving as a valuable metric for recruitment decisions.
    
    \item \textbf{Age Distribution Balance:} The squad age profile shows 70\% of players in the 21–30 range, representing an optimal mix of developing talent (21–25) and prime performers (26–30).
    
    \item \textbf{Right vs Left Foot Bias:} Right-footed goals dominate the scoring charts (Sergio Agüero: 127 right-foot goals vs Mohamed Salah: 63 left-foot goals), indicating tactical implications for defensive positioning.
    
    \item \textbf{Shooting Accuracy Variation:} Top shooting accuracy (54\%) demonstrates that precision is as important as volume when evaluating forward quality.
    
    \item \textbf{Assist Contribution:} Creative forwards like Theo Walcott (51 assists) provide significant value beyond goal scoring and should be evaluated holistically.
\end{enumerate}

\vspace{0.3cm}

\begin{tcolorbox}[colback=primary!10, colframe=primary, arc=5pt, boxrule=1.5pt]
\textcolor{primary}{\textbf{\faLightbulb\hspace{0.3cm} Final Recommendations:}}
\begin{itemize}
    \item Prioritize recruitment of players aged 21–25 with high shot conversion rates
    \item Use xG difference as a key metric for identifying undervalued talent
    \item Maintain balanced age distribution (70\% in 21–30 range) for squad stability
    \item Develop both left-foot and right-foot attacking options for tactical flexibility
    \item Monitor underperformers with negative xG difference for potential improvement or replacement
\end{itemize}
\end{tcolorbox}
